=== To Run ===

Backend:
  cd backend && venv\Scripts\activate && python main.py
  → http://localhost:8000 (API docs at /docs)

Frontend:
  cd frontend && npm run dev
  → http://localhost:5173

AI features: set LLM_API_KEY in backend/.env

All code is complete implementations (not stubs). The 3D cabinet renders with
5 body boards, click-to-select works, constraint solver enforces all 12 rules,
7 Skills handle natural language editing, and the Agent uses OpenAI function
calling to route intent to Skills with SSE streaming.
